Ex01 : Django, framework web.

Qu'est-ce que Django ?

Django est un framework web Python de haut niveau qui encourage le développement rapide et une conception propre et pragmatique. Créé par des développeurs expérimentés, il prend en charge une grande partie des tracas du développement web, vous permettant de vous concentrer sur l'écriture de votre application sans avoir à réinventer la roue.

Historique

Django a été créé à l'automne 2003, lorsque les développeurs web du journal Lawrence Journal-World ont commencé à utiliser Python pour créer des applications. Adrian Holovaty et Simon Willison, qui travaillaient au journal, ont décidé de factoriser leur code commun dans un framework web générique.

En juillet 2005, Django a été publié publiquement sous licence BSD. Le framework porte le nom du guitariste de jazz Django Reinhardt.

Philosophie

  • DRY (Don't Repeat Yourself) : Chaque morceau de connaissance doit avoir une représentation unique dans le système.
  • Batteries incluses : Django inclut de nombreux outils prêts à l'emploi.
  • Convention over configuration : Django fait des choix raisonnables par défaut.
